# Modelling-and-Simulation-of-Passive-Emergency-Heat-Removal-System-for-SMRs
Steady-state numerical model in Python to analyze the thermal-hydraulic performance of a Passive EHRS for a Gen-III+ Small Modular Reactor.  The project involved simulating a closed-loop natural circulation system, utilizing IAPWS97 formulations to model two-phase flow phenomena within a helical coil steam generator and an isolation condenser.
